# Astro Project

## Requirements

- [Node LTS](https://nodejs.org/)
- [PNPM](https://pnpm.io/)
- [Git](https://git-scm.com/)

## Starting the project

To start developing, run the following command:

```bash
pnpm dev
```

And open the [localhost:4321](localhost:4321) page.
